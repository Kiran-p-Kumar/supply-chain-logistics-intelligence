**Supply Chain Analytics Dashboard
**
Built a business-focused Supply Chain Analytics solution using Python, SQL, and Power BI to analyze supplier performance, inventory utilization, freight cost, shipment operations, and delivery lead times from global logistics data.

**Business Problem**

Manual analysis of logistics and shipment data made it difficult to track delivery delays, freight expenses, inventory distribution, and supplier efficiency.

**Solution**
1.Cleaned and transformed raw SCMS logistics data using Python & Pandas

2.Built supplier and inventory master tables

3.Performed SQL-based KPI analysis

4.Developed interactive Power BI dashboard for operational monitoring

**Key Insights**

1.Identified high freight cost shipment modes

2.Analyzed supplier lead time delays

3.Tracked warehouse inventory distribution

4.Monitored shipment and order fulfillment trends

**Business Impact**

Reduced manual reporting effort and improved visibility into logistics operations, inventory planning, and supplier performance through centralized analytics reporting.

**Tech Stack**

Python | Pandas | NumPy | MySQL | Power BI (DAX, Star Schema)
